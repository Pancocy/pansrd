基本数据类型 ​

Rust是一门静态类型语言,这意味着在编译期就需要知道所有值的类型。虽然在多数情况下编译器能够推测出我们想要使用的类型，但随着程序的庞大和复杂，变量的类型不确定性也就更高了，因此声明数据时进行类型标注是有必要的。这里主要介绍两大类数据类型：标量类型、复合类型。

1.标量类型 ​

标量标识单个值的概念。标量类型包括：整数类型、浮点数类型、布尔类型、字符类型。

1.1 整数类型 ​

整数类型是不包含小数部分的数字，整型还分为有符号和无符号类型，对应其在不同位宽的计算机上所占的字节数，构成一下情况。默认情况下，整数类型是32位无符号整数。

1.2 浮点数类型 ​

浮点数类型是包含小数部分的数字，分为f32和f64两种，f32是单精度浮点数，f64是双精度浮点数。默认情况下，浮点数类型是64位双精度浮点数。所有浮点型都是有符号的(insigned).

1.3 *数字运算 ​

Rust所有的数字类型都支持基本的数学运算，运算符包括加、减、乘、除、取模等。整数除法会向下取整。此外浮点数和整数之间可以进行运算，运算结果为浮点数。

1.4 布尔类型 ​

和大多数语言一样，布尔类型只有两个值：true和false。具有显示声明的bool类型：

rust
  let t = true
  
  let f:bool = false //显式声明

1.5 字符类型 ​

Rust 的 char（字符）类型是该语言最基本的字母类型。 char 字面量采用单引号括起来，字符串字面量是用双引号括起来，字符大小为4字节。

rust
fn main() {
    let c = 'z';
    let z = 'ℤ';
    let heart_eyed_cat = '😻';
}

2. 复合类型 ​

在Rust中，将多个值组合在一起可以组成复合类型，常见的复合类型又元组（tuple）和数组（array）

2.1 元组 ​

元组是Rust中一个非常特殊的复合类型，它允许我们将多个不同类型的值组合在一起。元组是固定长度的，一旦创建，元组中的元素数量和类型就不能改变。

1.创建元组：

rust
  let tupl:(i32,f64,u8) = (500,6.4,1);

2.访问元组。有两种方式访问元组中的元素:解构、索引。

rust
  let tupl:(i32,f64,u8) = (500,6.4,1);
  
  let (x,y,z) = tupl;
  println!("x = {},y = {},z = {}",x,y,z);
  
  println!("x = {},y = {},z = {}",tupl.0,tupl.1,tupl.2);

3.特殊的元组。没有值或者只有一个值的元组称为单元类型(unit type)，表示为()。该值被称为单元值（unit value）。如果表达式不返回任何其他值，就隐式地返回单元值。

2.2 数组 ​

数组是Rust中一个重要的复合类型，它允许我们将多个相同类型的值组合在一起。数组是固定长度的，一旦创建，数组中的元素数量和类型就不能改变。Rust标准库存在一种动态数组，称之为向量(vector)。如果不确定是使用数组还是 vector，那就应该使用一个 vector,不过当你明确元素数量不需要改变时，数组会更有用。

1.创建数组有很多方式：

rust
   //1.直接创建
   let names = ["Bob","Alice","Carol"];
   
   //2.标明类型。&str表示每个元素的类型,3表示数组的长度
   let names: [&str; 3] = ["Bob","Alice","Carol"];

   //3.当数组中的所有元素都相等时。3表示每个元素的值,5表示数组的长度
   let a = [3; 5];  // [3, 3, 3, 3, 3]

2.访问数组的元素 通过索引可以访问数组的元素，索引从0开始。当超出索引来访问时，会引起程序panic。

rust
  println!("The first element is {}",names[0]);
  println!("The second element is {}",names[1]);
  println!("The third element is {}",names[2]);
  println!("The fourth element is {}",names[3]); //panic
  
  for name in names{}
  for name in names.iter() {}
